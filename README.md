# 💼 JobMatch AI

JobMatch AI is a simple intelligent job recommendation app that helps users find jobs based on their **skills** and **experience level**.

Built with **Python** and **Streamlit**, this app analyzes your skills and shows how well you match different job roles.

---

## 🚀 Features

* 🛠️ Select your skills from a dynamic list
* 📊 Choose your experience level (Beginner / Intermediate / Advanced)
* 🎯 Smart job matching based on skill overlap
* 📈 Match percentage for each job
* ✅ Shows:

  * Matching skills
  * Missing skills (what to learn next)

---

## 🧠 How It Works

The app compares your selected skills with job requirements:

* Filters jobs based on your experience level
* Calculates:

  * Number of matching skills
  * Total required skills
  * Match percentage
* Displays results sorted by best match
